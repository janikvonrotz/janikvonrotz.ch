## Usage

Clone the repo.

```bash
git clone git@github.com:janikvonrotz/janikvonrotz.ch.git
cd janikvonrotz.ch
```

Install hugo.

```bash
task install
```

Build the site.

```bash
task build
```

Serve the output.

```bash
task serve
```

### Develop

Start the development server.

```bash
task dev
```
